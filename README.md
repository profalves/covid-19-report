# COVID-19 report

COVID-19 report app made with react

## Getting Started

### Prerequisites

Install all dependences:

```bash
yarn
# or npm i
```

### Development mode

```bash 
yarn start
# or npm start
```

### To create a production build

```bash
yarn build
# or npm run build
```
